My Projects

Photoshop ID Editing Exercise

An ID editing project completed during our UX Design class using Adobe Photoshop.

Photoshop ID Editing Exercise

This project was completed as part of our User Experience Design Fundamentals class. Using Adobe Photoshop, I practiced editing and manipulating an identification card by replacing the original information with my own. This exercise helped enhance my familiarity with Photoshop tools and layers.

Photoshop Makeup Editing

A facial enhancement activity using makeup tools in Adobe Photoshop.

Photoshop Makeup Editing

We used a no-makeup photo found online and applied various makeup elements using Adobe Photoshop. I explored digital blush, lipstick, and other retouching tools, improving my understanding of beauty editing techniques in graphic design.

Photo Restoration Project

A restoration of an old photo using Adobe Photoshop tools.

Photo Restoration Project

As a Photoshop exercise, I found an old, damaged image online and used restoration tools to enhance and repair it. I applied the healing brush, clone stamp, and other features to bring the photo back to life.

Manipulation Project – Final Output

A fantasy composition showing me sitting on a white wolf in winter.

Manipulation Project – Final Output

This was my final project for our UX Design Fundamentals class. The goal was to create a photo manipulation, and I designed an image of myself sitting on a white wolf in a snowy environment. It demonstrates my creativity and use of advanced layering techniques.

To-Do List App UI

For this class activity, I designed a simple and clean user interface for a to-do list mobile app using Lucid. The mockup includes home, details, and settings screens, focusing on usability and simplicity.
